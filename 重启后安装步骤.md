# 重启后安装步骤

## 前提条件
- 已重启系统（清除旧的服务标记）

## 安装步骤

### 方法 1：使用最新安装包（推荐）

1. **下载最新版本**
   - 访问：https://github.com/hicool-ml/tp-link-broadband-dialer/releases/tag/v2.1
   - 下载 `Setup.exe`

2. **卸载旧版本**（如果已安装）
   - 打开"控制面板" → "程序和功能"
   - 找到"宽带拨号"或"Broadband Dial Tool"
   - 右键 → 卸载

3. **安装新版本**
   - 右键 `Setup.exe` → 以管理员身份运行
   - 按照安装向导完成安装

4. **验证服务状态**
   - 打开命令提示符（管理员）
   - 运行：`sc query TPLinkShutdownCleanup`
   - 应该看到 `STATE: RUNNING`

### 方法 2：手动安装服务

如果安装包的服务安装失败，可以手动安装：

1. **打开命令提示符（管理员）**

2. **导航到安装目录**
   ```cmd
   cd "C:\Program Files (x86)\TPLinkDialer"
   ```

3. **安装服务**
   ```cmd
   CleanupService.exe install
   ```

4. **启动服务**
   ```cmd
   sc start TPLinkShutdownCleanup
   ```

5. **验证状态**
   ```cmd
   sc query TPLinkShutdownCleanup
   ```

## 验证安装

### 检查服务状态
```cmd
sc query TPLinkShutdownCleanup
```

应该显示：
```
STATE: 4 RUNNING
```

### 查看服务日志
```cmd
type %TEMP%\tplink_cleanup\cleanup_service.log
```

### 使用 ServiceInstaller
1. 运行：`C:\Program Files (x86)\TPLinkDialer\ServiceInstaller.exe`
2. 查看服务状态

## 故障排除

### 服务无法启动

**问题**：错误 1053（服务未及时响应）

**解决**：
1. 检查配置文件是否存在：
   ```cmd
   type %USERPROFILE%\.tplink_dialer\config.json
   ```

2. 如果配置文件不存在，运行主程序创建配置：
   - 启动：`C:\Program Files (x86)\TPLinkDialer\TP-Link_Dialer.exe`
   - 在配置向导中输入路由器信息

3. 重新启动服务

### 服务已安装但停止

**问题**：服务显示 STOPPED

**解决**：
```cmd
sc start TPLinkShutdownCleanup
```

### 找不到服务

**问题**：`sc query` 返回"指定的服务不存在"

**解决**：
1. 检查是否在正确的目录
2. 以管理员身份运行命令
3. 如果仍然失败，重启系统后再试
