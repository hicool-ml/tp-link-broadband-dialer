# 宽带拨号工具 v2.1 - 快速入门

**最后更新**: 2026-03-19

---

## 🚀 30秒快速上手

### 1️⃣ 安装程序
```
双击 Setup.exe → 一键安装
```

### 2️⃣ 配置路由器（首次运行）
```
启动程序 → 输入路由器IP和管理密码 → 点击保存
```

### 3️⃣ 开始拨号
```
输入宽带账号密码 → 点击"开始连接" → 完成！
```

---

## 📋 系统要求

- Windows 7/10/11
- 1GB 可用磁盘空间
- 管理员权限（安装服务时）

---

## 🔧 主要功能

| 功能 | 说明 |
|------|------|
| **一键拨号** | 自动完成 PPPoE 拨号 |
| **自动清理** | 关机时自动清除账号密码 |
| **配置保存** | 密码加密存储，下次使用无需输入 |
| **系统托盘** | 最小化到托盘，不占任务栏 |

---

## ⚙️ 常用操作

### 修改路由器配置

1. 启动程序
2. 点击右下角"⚙ 设置"按钮
3. 修改配置并保存

### 查看运行日志

双击进度条区域

### 系统托盘

- 最小化到托盘：点击关闭按钮
- 恢复窗口：双击托盘图标

- ✅ 密码只保存在内存中
- ✅ 关机时自动清理路由器账号
- ✅ 配置文件加密存储
