# TP-Link 宽带拨号工具

一个用于自动管理TP-Link路由器宽带拨号的Python工具，支持自动连接、断开和清除账号密码，确保账号安全。

## 功能特点

### 核心功能
- ✅ **自动拨号**：自动登录路由器并配置宽带账号密码
- ✅ **自动断开**：自动断开连接并清除账号密码
- ✅ **账号安全**：关机前自动执行清理操作，防止账号泄露
- ✅ **系统托盘**：最小化到系统托盘，双击可显示窗口
- ✅ **状态监控**：实时显示连接状态和运行日志

### 安全特性
- ✅ **关机拦截**：检测到关机时自动阻止并执行清理
- ✅ **自动清理**：使用完成后自动清除账号密码
- ✅ **验证机制**：验证账号密码是否完全清除
- ✅ **日志记录**：详细记录所有操作，便于追踪

### 用户体验
- ✅ **一键连接**：输入账号密码后点击连接即可
- ✅ **自动聚焦**：启动时光标自动定位到账号输入框
- ✅ **快捷键支持**：支持回车键快速切换和连接
- ✅ **进度显示**：实时显示操作进度和状态
- ✅ **调试窗口**：双击进度条可查看详细日志

## 关机拦截原理

### 核心原理

Windows 关机时会向所有窗口发送 `WM_QUERYENDSESSION` 消息，程序返回 `0`（False）会让系统弹出"该程序正在使用中，是否强制关机"的提示框。

**重要说明：**
- 这是系统层面能争取的最大权限
- 无法 100% 阻止关机，但能提醒用户
- 用户最终仍可选择"强制关机"

### 实现方式

1. **接收系统消息**：通过 Tkinter 的 `WM_QUERYENDSESSION` 协议监听关机事件
2. **检测账号状态**：检查是否有保存的账号密码
3. **阻止关机**：返回 `False` 阻止关机，系统会弹出提示框
4. **执行清理**：自动执行断开和清除操作
5. **释放阻止**：清理完成后允许系统继续关机

### 使用条件

- ✅ 需要 Python 3.x 版本
- ✅ 仅支持 Windows 系统（macOS/Linux 实现方式不同）
- ✅ 无需额外安装库（tkinter 和 ctypes 都是内置的）

### 测试效果

1. **运行程序后直接点 Windows 关机按钮**
   - 会先弹出"未断网"的提醒框
   - 关闭提醒后系统会再弹出"强制关机"的确认框

2. **先在终端按 Ctrl+C 退出程序**
   - 程序会自动断网
   - 此时再关机就不会触发提醒

### 跨平台说明

- **Windows**：监听 `WM_QUERYENDSESSION` 消息（已实现）
- **macOS**：监听 SIGTERM 信号（需适配）
- **Linux**：监听 shutdown 事件（需适配）

如需适配其他平台，请联系开发者。

## 使用方法

### 1. 基本使用

1. 运行程序：
   ```bash
   python tp_link_broadband_dialer.py
   ```

2. 输入宽带账号和密码

3. 点击"开始连接"按钮

4. 等待连接成功，程序会自动隐藏到系统托盘

5. 使用完成后，点击"断开连接"按钮清除账号密码

### 2. 系统托盘操作

- **双击托盘图标**：显示程序窗口
- **右键菜单**：
  - 显示窗口
  - 隐藏窗口
  - 断开并清除

### 3. 关机保护

当系统关机时，如果程序检测到账号密码仍保存在路由器中：
1. 程序会自动阻止关机
2. 显示警告对话框
3. 自动执行断开和清除操作
4. 显示清理结果
5. 允许系统继续关机

## 技术架构

### 核心技术

- **Playwright**：用于自动化浏览器操作
- **Tkinter**：GUI界面框架
- **pystray**：系统托盘功能
- **ctypes**：Windows API调用

### 关机拦截实现

使用Windows API实现真正的关机拦截：

1. **SetProcessShutdownParameters**：设置关机参数
   - 最高优先级（0x3FF）
   - 不重试标志（SHUTDOWN_NORETRY）

2. **ShutdownBlockReasonCreate**：创建关机阻止
   - 阻止系统关机
   - 显示阻止原因

3. **ShutdownBlockReasonDestroy**：释放关机阻止
   - 清理完成后释放
   - 允许系统继续关机

## 注意事项

1. **管理员密码**：确保路由器管理员密码正确
2. **网络连接**：确保计算机能够访问路由器管理界面

4. **手动清理**：如果自动清理失败，请手动点击"断开连接"按钮
