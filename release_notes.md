## 宽带拨号 v2.1

### 主要特性
- 零依赖运行 - 无需安装 Python、浏览器
- 一键连接 - 自动完成 PPPoE 拨号
- 自动清理 - 关机时清除路由器账号密码
- 后台服务 - Windows 服务自动清理
- 配置管理 - 图形化配置向导

### v2.1 修复内容
- 修复服务安装路径问题
- 添加桌面快捷方式图标
- 快捷方式中文化
- 修复服务启动超时问题
- 添加缺失的模块
- 延迟初始化避免阻塞
- 安装包体积减少 72%

### 安装说明
1. 下载 Setup.exe
2. 右键以管理员身份运行
3. 按向导完成安装

### 重要提示
如果遇到服务启动失败:
1. 重启系统
2. 手动安装服务

### 系统要求
- Windows 7 SP1 或更高版本
- 管理员权限
