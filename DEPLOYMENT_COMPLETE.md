## 🚀 部署方式

### 方式1：绿色版部署（推荐）

#### 步骤：

1. **打包发布目录**
   ```bash
   cd Release
   zip -r 宽带拨号工具_v2.0.zip .
   ```

2. **分发给用户**
   - 发送 `宽带拨号工具_v2.0.zip`
   - 大小约310MB

3. **用户安装**
   - 解压到任意目录
   - 右键 `一键安装.bat` → 以管理员身份运行
   - 从桌面快捷方式启动

#### 优点：
- ✅ 无需安装程序
- ✅ 可以直接部署
- ✅ 便于分发和更新

### 方式2：NSIS安装包

#### 需要准备：

1. **安装NSIS**
   - 下载：https://nsis.sourceforge.io/Download
   - 安装后确保 `makensis` 命令可用

2. **创建安装包**
   ```bash
   makensis SetupScript.nsi
   ```
   输出：`Release/Setup.exe`

3. **分发给用户**
   - 发送 `Setup.exe`
   - 大小约310MB（压缩后更小）

#### 优点：
- ✅ 专业安装体验
- ✅ 自动注册表管理
- ✅ 自动卸载支持

## 📋 用户端部署步骤

### 第一次安装

1. **管理员权限**
   ```
   右键 "一键安装.bat" → 以管理员身份运行
   ```

2. **自动完成**
   - ✅ 安装清理服务
   - ✅ 启动清理服务
   - ✅ 创建桌面快捷方式

3. **验证安装**
   ```bash
   # 检查服务状态
   CleanupService\CleanupService.exe status

   # 或使用图形界面
   ServiceInstaller.exe
   ```

### 日常使用

1. **启动程序**
   - 双击桌面 "宽带拨号工具" 图标
   - 或运行 `TP-Link_Dialer\TP-Link_Dialer.exe`

2. **连接拨号**
   - 输入账号密码
   - 点击 "开始连接"

3. **断开连接**
   - 点击 "断开连接" 清除账号
   - 或直接关闭（关机时服务自动清理）

### 卸载程序

1. **管理员权限**
   ```
   右键 "一键卸载.bat" → 以管理员身份运行
   ```

2. **手动清理**
   - 删除程序目录

## ⚠️ 注意事项

### 用户端要求

1. **管理员权限**
   - 安装服务需要管理员权限
   - 首次运行需要管理员身份

2. **系统兼容性**
   - Windows 7 SP1 或更高版本
   - 推荐 Windows 10/11

3. **磁盘空间**
   - 至少 500MB 可用空间

### 安装注意事项

1. **杀毒软件**
   - 可能被误报为病毒
   - 需要添加到信任列表

2. **防火墙**
   - 无需特殊配置
   - 程序连接本地路由器

3. **网络要求**
   - 需要连接到路由器
   - 路由器IP: 192.168.1.1
