## 🚀 用户端部署

### 安装步骤

1. **解压发布包**
   ```
   将 宽带拨号工具_v2.0_安装包.zip 解压到任意目录
   ```

2. **运行安装脚本**
   ```
   右键 "一键安装.bat" → 选择 "以管理员身份运行"
   ```

3. **安装完成**
   - ✅ 清理服务已安装并启动
   - ✅ 桌面快捷方式已创建
   - ✅ 开始菜单已添加

4. **启动程序**
   ```
   双击桌面 "宽带拨号工具" 图标
   ```

### 使用流程

1. **输入账号密码**
2. **点击"开始连接"**
3. **等待连接成功**
4. **使用网络**
5. **使用完成后**:
   - 手动点击"断开连接"，或
   - 直接关闭程序（关机时服务自动清理）

## 🐛 故障排除

### 常见问题

**Q: 主程序无法启动**
- 检查杀毒软件是否拦截
- 以管理员身份运行
- 查看Windows事件查看器

**Q: 服务无法安装**
- 确保以管理员身份运行
- 检查PyWin32是否正确打包
- 查看服务日志

**Q: 关机时账号未清除**
- 检查服务是否正在运行
- 确保正常关机（不要强制关机）
- 查看服务日志确认清理过程
