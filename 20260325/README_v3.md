# TP-Link 宽带拨号助手 v3.0.0

## 🎉 全新 RunOnce 机制

### ✨ 核心特性

- ✅ **自动拨号** - TP-Link 路由器 PPPoE 自动拨号
- ✅ **MAC 设置** - 支持路由器MAC/PC MAC/自定义MAC
- ✅ **自动清理** - 关机/重启/注销后自动清理路由器账号密码
- ✅ **100%覆盖** - 采用微软 RunOnce 机制，保证下次启动一定清理

## 🚀 快速开始

### 安装

1. **双击安装程序**
   ```
   Broadband_Dialer_Setup_v3.0.0.exe
   ```

2. **按提示安装**
   - 欢迎 → 阅读许可协议 → 选择目录 → 安装 → 完成

3. **安装完成后**
   - 桌面会出现"宽带连接"快捷方式
   - RunOnce 清理机制已自动激活

### 使用

1. **启动程序**
   - 双击桌面"宽带连接"快捷方式
   - 或从开始菜单启动

2. **配置路由器**
   - 首次使用会打开设置对话框
   - 输入路由器 IP 地址（默认：192.168.1.1）
   - 输入路由器管理密码

3. **开始拨号**
   - 输入宽带账号
   - 输入宽带密码
   - 点击"开始连接"
   - 等待连接成功（约10-15秒）

4. **最小化到托盘**
   - 连接成功后程序最小化到系统托盘
   - 双击托盘图标可重新显示窗口

## 🎯 RunOnce 清理机制

### 工作原理

```
关机/重启/注销 → 自动注册 RunOnce → 下次开机自动清理路由器
```

### 触发时机

| 场景 | 是否清理 | 说明 |
|------|---------|------|
| 正常关机 | ✅ | 注册 RunOnce，下次开机清理 |
| 正常重启 | ✅ | 注册 RunOnce，下次开机清理 |
| 用户注销 | ✅ | 注册 RunOnce，下次开机清理 |
| 强制断电 | ✅ | 注册 RunOnce，下次开机清理 |
| 蓝屏崩溃 | ✅ | 注册 RunOnce，下次开机清理 |

**只要机器能再开机，就一定清理！**

### 日志位置

```
C:\ProgramData\BroadbandDialer\cleanup.log         - PowerShell 日志
C:\ProgramData\BroadbandDialer\cleanup_http.log    - 清理程序日志
```

## ❓ 常见问题

### Q: 为什么重启后路由器还在拨号？

A: RunOnce 机制在**下次开机时**执行清理，不是立即清理。

**验证方法**：
1. 检查日志：`C:\ProgramData\BroadbandDialer\cleanup_http.log`
2. 如果有"清理完成"日志，说明已执行
3. 访问路由器管理界面，检查账号密码是否已清空

### Q: 如何确认 RunOnce 已注册？

A: 打开注册表编辑器，查看：
```
HKLM\Software\Microsoft\Windows\CurrentVersion\RunOnce\TPLinkCleanup
```

如果存在，说明已注册。

### Q: 安装后立即重启，会清理吗？

A: **会！** 安装程序已自动注册 RunOnce，下次开机就会清理。

### Q: 卸载后 RunOnce 会删除吗？

A: **会！** 卸载程序会自动删除 RunOnce 注册表项。

### Q: 日志文件会占用很多空间吗？

A: 不会。日志文件很小（几KB），而且可以随时删除。
