## 项目概述

**宽带拨号工具** 是一个用于管理 TP-Link 路由器宽带拨号的 Windows 工具，支持自动连接和关机时自动清理账号密码。

### 主要特点

- ✅ **零依赖运行** - 无需安装 Python、浏览器或任何运行环境
- ✅ **一键连接** - 自动完成 PPPoE 拨号配置
- ✅ **自动清理** - 关机时自动清除路由器账号密码
- ✅ **配置管理** - 图形化配置向导，密码加密存储
- ✅ **智能清理** - 检查路由器状态，按需清理
- ✅ **完整卸载** - 一键卸载，保护用户数据

## 后台清理服务

### 🎯 服务功能

**服务名称**: `TPLinkShutdownCleanup`
**显示名称**: `TP-Link路由器账号清理服务`

**功能**: 在系统关机时自动清除路由器中的宽带账号密码

### 📋 服务管理

#### 查看服务状态

```batch
# 方法 1: 使用服务管理工具
C:\Program Files\TPLinkDialer\ServiceInstaller.exe

# 方法 2: 命令行
sc query TPLinkShutdownCleanup
```

#### 启动/停止服务

```batch
# 启动服务
net start TPLinkShutdownCleanup

# 停止服务
net stop TPLinkShutdownCleanup
```

### 🔍 智能清理机制

```
关机时自动执行：

1. 检查路由器连接状态
2. 检查账号是否已配置
3. 如果已断开且无账号 → 跳过清理
4. 如果已连接或有账号 → 执行清理
   - 停止拨号连接
   - 清空账号密码
   - 保存配置
   - 验证清理成功
```

### 📊 服务日志

```
日志位置:
%TEMP%\tplink_cleanup\cleanup_service.log

查看方法:
1. 打开日志文件
2. 或使用服务管理工具的"查看服务日志"功能
```

## 卸载说明

### 🗑️ 卸载方法

#### 方法 1: 控制面板（推荐）

```
1. 打开"控制面板"
2. 点击"程序和功能"（或"卸载程序"）
3. 找到"Broadband Dial Tool"
4. 右键点击 → 选择"卸载"
5. 确认卸载
```

#### 方法 2: 运行卸载程序

```
1. 打开安装目录:
   C:\Program Files\TPLinkDialer\

2. 双击运行:
   uninstall.exe

3. 确认卸载
```

#### 方法 3: 命令行卸载

```batch
# 正常卸载
"C:\Program Files\TPLinkDialer\uninstall.exe"

# 静默卸载（无提示）
"C:\Program Files\TPLinkDialer\uninstall.exe" /S
```

### 📋 卸载流程

卸载程序会自动执行以下操作：

```
[步骤 1/6] 停止清理服务
  → 停止 TPLinkShutdownCleanup 服务

[步骤 2/6] 移除 Windows 服务
  → 卸载清理服务

[步骤 3/6] 停止运行进程
  → 终止所有相关程序

[步骤 4/6] 删除卸载程序
  → 删除 uninstall.exe 本身

[步骤 5/6] 删除程序文件
  → 删除整个安装目录

[步骤 6/6] 清理注册表
  → 删除注册表项
```

### 🔒 用户数据保护

**保留内容**（不会被删除）:
- ✅ 配置文件: `%USERPROFILE%\.tplink_dialer\config.json`
- ✅ 服务日志: `%TEMP%\tplink_cleanup\cleanup_service.log`

**删除内容**:
- ✅ 程序文件: `C:\Program Files\TPLinkDialer\`
- ✅ Windows 服务: `TPLinkShutdownCleanup`
- ✅ 桌面快捷方式
- ✅ 开始菜单项
- ✅ 注册表项

## 故障排除

### ❓ 主程序无法启动

#### 问题：双击没反应

**可能原因**:
- 杀毒软件拦截
- 程序已在运行
- 缺少运行库

**解决方法**:
1. 检查杀毒软件是否拦截
2. 检查任务管理器中是否已运行
3. 以管理员身份运行
4. 重新安装程序

#### 问题：提示缺少浏览器

**可能原因**:
- 安装不完整

**解决方法**:
1. 重新运行安装包
2. 选择"修复"或"重新安装"

---

### ❓ 无法连接路由器

#### 问题：连接失败

**可能原因**:
- 路由器IP地址错误
- 路由器密码错误
- 网络线路故障
- 路由器故障

**解决方法**:
1. 检查路由器IP配置是否正确
2. 验证路由器管理密码
3. 检查网线连接
4. 查看详细日志（双击进度条）

#### 问题：获取IP地址失败

**可能原因**:
- 宽带账号或密码错误
- 宽欠费
- 线路故障

**解决方法**:
1. 验证宽带账号密码
2. 检查是否欠费
3. 联系运营商
4. 重新尝试连接

---

### ❓ 服务无法安装

#### 问题：安装失败

**可能原因**:
- 权限不足
- 服务已存在
- PyWin32 问题

**解决方法**:
1. 以管理员身份运行
2. 卸载旧版本服务后重试
3. 检查服务日志

#### 问题：服务无法启动

**可能原因**:
- 配置文件未找到
- 浏览器组件缺失

**解决方法**:
1. 检查配置文件是否存在
2. 重新安装完整版
3. 查看服务日志

---

### ❓ 关机时账号未清除

#### 问题：账号密码仍然存在

**可能原因**:
- 服务未运行
- 服务配置错误
- 强制关机

**解决方法**:
1. 检查服务状态:
   ```batch
   sc query TPLinkShutdownCleanup
   ```

2. 查看服务日志:
   ```
   %TEMP%\tplink_cleanup\cleanup_service.log
   ```

3. 确保正常关机（不要强制关机）
